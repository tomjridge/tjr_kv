Module Kv_intf.Syncable_map

type ('k, 'v, 'ptr, 't) syncable_map = {
find : 'k -> ('v option, 't) Tjr_monad.m;
insert : 'k -> 'v -> (unit, 't) Tjr_monad.m;
delete : 'k -> (unit, 't) Tjr_monad.m;
sync : unit -> ('ptr, 't) Tjr_monad.m;
ksync : 'k -> ('ptr, 't) Tjr_monad.m;
ks_sync : 'k list -> ('ptr, 't) Tjr_monad.m;
}

A syncable map API.

Sync operations:

  • sync to sync the whole map to disk
  • ksync to sync a particular key
  • ks_sync to sync a set of keys
